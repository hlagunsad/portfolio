# Harry Lagunsad — Portfolio

Personal portfolio site for a full-stack / frontend engineer.

**Live:** https://portfolio-three-theta-72uldwcq7e.vercel.app

## Tech
- Next.js 16 (App Router, Server Components) · React 19 · TypeScript
- Tailwind CSS v4
- Fully static, deployed on Vercel

## Run locally
```bash
npm install
npm run dev     # http://localhost:3000
npm run build   # production build
```

## Editing
- Content (projects, skills, experience): `src/lib/data.ts`
- Theme: the eight color tokens at the top of `src/app/globals.css`
